# BeatLab---Beta-V1.1
BeatLab is a locally hosted AI music-drum pattern generator that has to ability to generate drum patterns in 3 different genres (Hip-Hop, Pop, and Rock). This model runs using Ollama llama3. User can describe the desired drum pattern to their custom demands. Melody Pattern Generation Coming in Beta Version 2.0

ADDED:

1. 2 new Genres for Drum Pattern Generation
2. Ollama llama3 WebUI
3. MIDI File Exports
